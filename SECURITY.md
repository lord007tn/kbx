# Security Policy

## Supported Versions

We currently support the latest `0.1.x` release line.

## Reporting A Vulnerability

Please do not open public issues for security vulnerabilities.

Use GitHub Security Advisories for this repository:

https://github.com/lord007tn/kbx/security/advisories/new

When reporting, include:

1. A clear description of the issue and impact.
2. Reproduction steps or proof-of-concept.
3. Affected versions or commits.
4. Suggested mitigations if available.

We will acknowledge reports as quickly as possible and coordinate a fix and disclosure timeline.
