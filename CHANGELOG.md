## 0.1.0 - 2026-05-19

Initial alpha release candidate for the public npm package.

### Added

- Open-source readiness docs, release checklist, root maintainer file, CODEOWNERS, and CodeQL workflow.
- Project-local `changelogithub` release script and dependency override to keep npm audit clean.
- Local-first `kbx` CLI for workspace init, ingest, search, context, status, doctor, config, model management, watch mode, and stats.
- Stdio MCP server with search, context, file-context, graph, session, memory, maintenance, and gated destructive tools.
- Local vector and lexical retrieval with branch-aware indexing, chunk dedupe, optional graph expansion, reranking, and cited result previews.
- Ingest support for Markdown, plain text, common source files, PDF, DOCX, PPTX, XLSX, EPUB, image metadata/OCR, and structured text formats.
- npm package verification, install smoke tests, CI, release automation, issue templates, contribution docs, security policy, support policy, and code of conduct.

### Notes

- `kbx` is alpha software. Public API, command output, and package contents may change before v1.
- npm/npx is the supported distribution path for this alpha.
